레코드 시스템

진실의 근원(source of truth). 원천 데이터.

사용자 input으로 받은 데이터를 저장한것.

(일반적으로)정규화를 거쳐 저장함.

파생 데이터 시스템

다른 시스템에 있는 데이터를 가져와서 특정 방식으로 변환하고 처리한거.

데이터를 잃게되더라도 원천데이터로부터 다시 생성 가능
